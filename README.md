MEMZ Python Remix

A Python recreation of the visual and prank effects inspired by MEMZ.

All destructive functionality has been removed.

This project is intended for educational purposes, experimentation, and entertainment. It focuses on visual effects, cursor movement, message boxes, sound effects, and other harmless MEMZ-style behaviors.

---

Features

- Screen distortion effects
- Tunnel effect
- Random warning/error icons
- Screen color inversion
- Cursor movement effects
- Random message boxes
- Random Windows system sounds
- Random meme website opening

---

Important Notice

This is NOT the original MEMZ malware.

The following destructive behaviors have been removed:

- File deletion
- MBR modification
- Registry destruction
- Data corruption
- Self-replication
- System damage
- Any other destructive payloads

However, the program may:

- Move your mouse cursor
- Open browser tabs
- Display multiple message boxes
- Distort your screen temporarily

For this reason, running the program inside a virtual machine or test environment is recommended.

---

Requirements

- Python 3.x
- pywin32

Download Python:

https://www.python.org/downloads/

Install pywin32:

pip install pywin32

Running

python main.py
